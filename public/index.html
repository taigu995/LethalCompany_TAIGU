QuickOrganization - 飞船物品整理

v1.0.3

下载 QuickOrganization.dll

安装说明：

1. 将 QuickOrganization.dll 放入 BepInEx/plugins/ 目录

2. 启动游戏，按 = 键打开/关闭整理菜单

3. 配置文件: BepInEx/config/TAIGU.QuickOrganization.cfg

v1.0.3 更新：

- 修复 DiscardHeldObject 补丁目标类错误

- 修复键盘检测使用传统 Input 系统

- 添加 V81 SetParent 前缀补丁

- 署名改为 TAIGU

CleanShip - 飞船清洁模组

v1.2.0

下载 CustomCompany.CleabShip.dll 下载 LitJSON.dll (依赖库)

安装说明：

1. 将 CustomCompany.CleabShip.dll 放入 BepInEx/plugins/ 目录

2. 将 LitJSON.dll 放入游戏根目录（与 Lethal Company.exe 同级）

3. 启动游戏，配置文件将自动生成在 BepInEx/config/TAIGU.CleanShip.cfg

MicMusicPlayer - 麦克风音乐播放器

v1.0.0

下载 MicMusicPlayer.dll

安装说明：

1. 将 MicMusicPlayer.dll 放入 BepInEx/plugins/ 目录

2. 将音乐文件（MP3/OGG/WAV）放入 BepInEx/plugins/MicMusicPlayer/music/ 目录

3. 启动游戏，按 F2 或 = 键打开/关闭音乐播放器

功能特性：

- 使用 uGUI (Canvas) 渲染，兼容 URP 管线

- 支持麦克风混音（游戏内语音 + 音乐同步传输）

- 支持播放列表、音量调节、进度控制

- 通过 Windows API (GetAsyncKeyState) 检测按键，兼容新 Input System
